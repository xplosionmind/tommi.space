## Current data

Following advice gathered from [this post]() on the YunoHost forum, I found [btop++](https://github.com/aristocratos/btop) to be the best tool for observing the metrics of a system at present time. It is a simple solution to see at a glance how loaded the system is **while it’s running**.

I am interested in learning how to use common tools like [Prometheus](https://prometheus.io/) and [Grafana](https://grafana.com/) to monitor the system over an extended timespan, and potentially alerting me via email for warnings or errors.
